
La aplicación cuenta con dos microadministradores
muy funcionales dependiendo lo que se requiera,
para consulta rapida y ejecución de queries:

<host>/phpminiadmin.php con password "pass"

Para edicion de registros y exportación a csv
se cuenta con el otro:

<host>/libs/adminer pide los datos de la conexion.

los scripts y la documentacion se encuentran en la
carpeta doc, la última base se encuentra en
/doc/base.sql.gz

La parte de los correos es pesada ~3 MB pero
ya se mandaron cerca de 60 correos con el usuario
xptele2013@gmail.com el cual ademas guarda una
copia del correo enviado.
